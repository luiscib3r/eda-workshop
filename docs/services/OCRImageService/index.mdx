## Overview

El **OCR Image Service** es un worker que reacciona cuando un nuevo archivo es subido. Descarga el PDF desde S3, lo procesa utilizando `muPDF` para renderizar cada página como una imagen PNG, y sube estas imágenes de vuelta a S3.

### Flujo

1. Escucha `FileUploadedEvent`.
2. Descarga el PDF de S3.
3. Itera sobre cada página del PDF.
4. Renderiza la página como imagen PNG.
5. Sube la imagen a S3.
6. Publica `FilePageRenderedEvent` por cada página.
