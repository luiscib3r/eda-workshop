## Overview

El dominio de **OCR** (Optical Character Recognition) es el núcleo del procesamiento del sistema. Se encarga de transformar los archivos PDF en imágenes individuales, gestionar el flujo de procesamiento de cada página y utilizar inteligencia artificial para extraer el texto.

### Responsabilidades

- Renderizado de páginas de PDF a imágenes.
- Orquestación del flujo de procesamiento por página.
- Extracción de texto mediante LLMs.
- Almacenamiento de resultados de OCR.
